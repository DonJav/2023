Downloads

Nelson comes in two parts: the server and the client. Most users will be simply using a Nelson server that was setup by an operator in your organization, so the client will be what most people want to download.

Client

The client is shipped as a static binary that comes pre-compiled for a variety of operating systems. The tables below enumerate the direct-download sites for the latest version (presently v0.16.185).
